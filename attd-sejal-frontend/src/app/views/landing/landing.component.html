Being an online student, I have seen that attendance marking for our teachers in a classroom during a lecture is not only a onerous task but also a time consuming one. Due to an unusually high number of students present during the lecture there will always be a probability of proxy attendance(s).

So, with the help of:

CI/CD - GitHub Actions Azure - Azure App Service Angular Python Flask APIs Image Processing Azure Database for PostgreSQL flexible server Docker based containerization Token based Authentication

I created this to help her and many more!

Top Notch Application

The system works on face recognition where each student in the class is photographed, and their details are stored on a server. The teacher can record the attendance automatically in the classroom. The system will recognise the faces and verify the presence or absence of each student.

How to use?

Only 3 simple steps to follow:

  • Register yourself

  • Provide the detail of your students

  • Provide the link to your class
